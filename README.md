# BunnyPayPIM

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.2.0.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.


## Description 

Dedicated platform for the trader that allows stock management and shop management as well as a cash register with NFC technology.

Also administrative parts that generate recharge tickets, confirm the registration of a child and assign him an nfc tag.
